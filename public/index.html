Two-Tier Agent Architecture

The DynamicAgent routes user instructions over WebSockets and orchestrates Durable Object specialists via the MCP protocol. Each specialist exposes curated tools for its domain.

Production-Ready Workflows

Fork repositories, generate delivery plans, perform GitHub research, and log debugging sessions with guardrails built for autonomous agent execution.

How to get started

  1. Configure Durable Object, D1, KV, and AI bindings in wrangler.toml.
  2. Run npx wrangler d1 execute <database> --file=schema.sql or call /api/admin/init-db to initialise the database.
  3. Visit the Fork or Research pages to trigger the new workflows, or use Chat for raw WebSocket access.
